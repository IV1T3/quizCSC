Erklärung der Funktionen

init()

Aufruf: init()

Erklärung: Die Funktion init() muss IMMMER an den Anfang in den Code eingefügt werden. Sie bewirkt, dass die Engine startet und bei dem Programmstart ein Fenster erstellt wird. Ohne sie geht nichts!

draw()

Aufruf: draw()

Erklärung: Die Funktion draw() muss IMMMER nach allen Funktionen in den Code eingefügt werden. Sie bewirkt, dass alle beschriebenen Dinge auf den Bildschirm gezeichnet werden.

drawWords()

Aufruf mit Parametern/Argumenten: drawWords(Position, Text, Schriftgröße, Schriftfarbe)

Beispiel: drawWords((0,0), "Hallo", 23, (255,255,255))

Erklärung: Die Funktion drawWords() ermöglicht es einem, Text auf dem Bildschirm anzuzeigen. In diesem Beispiel wird "Hallo" oben links in der Bildschirm-Ecke in Schriftgrösse 23 in weiß ausgegeben.

drawCircle()

Aufruf mit Parametern/Argumenten: drawCircle(Mittelpunkt, Radius, Farbe)

Beispiel: drawCircle((50,50), 5, (0, 0, 255))

Erklärung: Die Funktion drawCircle() ermöglicht es einem, einen Kreis auf den Bildschirm zu zeichen. In diesem Beispiel wird ein blauer Kreis mit dem Mittelpunkt auf den Koordinaten (50, 50) und einem Radius von 5 Pixeln gezeichnet(10 Pixel Durchmesser).

drawEllipse()

Aufruf mit Parametern/Argumenten: drawEllipse(Start-x-Position, Start-y-Position, Breite, Höhe, Farbe)

Beispiel: drawEllipse(0, 0, 500, 500, (255,255,0))

Erklärung: Die Funktion drawEllipse() ermöglicht es einem, eine Ellipse auf den Bildschirm zu zeichen. In diesem Beispiel wird zuerst ein Viereck mit den Koordinaten x1->0, y1->0, x2->500, y2->500 erstellt, woraus anschließend eine Ellipse erstellt wird. Die Ellipse ist gelb.

drawLine()

Aufruf mit Parametern/Argumenten: drawLine(1. Punkt, 2. Punkt, Farbe)

Beispiel: drawLine((0,0), (1440, 900), (255,0,0))

Erklärung: Die Funktion drawLine() ermöglicht es einem, eine Linie auf den Bildschirm zu zeichen. In diesem Beispiel wird eine rote Linie von ganz oben links bis nach ganz rechts unten gezogen(nur bei einem Bildschirmformat von 1440x900)

drawRect()

Aufruf mit Parametern/Argumenten: drawRect(Start-x-Position, Start-y-Position, Breite, Höhe, Farbe)

Beispiel: drawRect(0, 0, 500, 500, (0, 255, 0))

Erklärung: Die Funktion drawRect() ermöglicht es einem, ein Viereck auf den Bildschirm zu zeichen. In diesem Beispiel wird ein grünes Quadrat von ganz oben links bis zu den Koordinaten x2->500, y2->500 gezeichnet.
