Allgemeines

Was ist QuizFL?

FL steht für FunctionsLibrary, also eine Zusammenstellung von Funktionen. Mit QuizFL kann man verschiedenste Objekte mit Hilfe von Python und PyGame auf ein generiertes Fenster zeichnen lassen.

Was ist RGB?

Erklärung: RGB ist die Abkürzung für Rot-Grün-Blau! Es können Werte von 0 bis 255 angegeben werden. Je höher die Zahl, desto heller die Farbe. Durch das Vermischen dieser 3 Farben können alle beliebigen Farben erstellt werden.

Farbe RGB Beispiel
Weiß (255,255,255)
Schwarz (0,0,0)
Rot (255,0,0)
Gelb (255,255,0)
Pink (255,0,255)

Erklärung der Funktionen

init()

Aufruf mit Parametern/Argumenten: init(BildschirmBreite, BildschirmHöhe, Vollbild, Hintergrundfarbe)

Beispiel: init( 500 , 500 , True , ( 0 , 0 , 255 ) )

Erklärung: Die Funktion init() muss IMMER an den Anfang in den Code eingefügt werden. Sie bewirkt, dass bei dem Programmstart ein Fenster erstellt wird. In diesem Beispiel würde ein 500x500 großes, blaues Fenster im Vollbildmodus gestartet werden.

draw()

Aufruf: draw()

Erklärung: Die Funktion draw() muss IMMER nach allen Funktionen an das Ende der While-True-Schleife eingefügt werden. Sie bewirkt, dass alle beschriebenen Dinge auf den Bildschirm gezeichnet werden.

destruct()

Aufruf: destruct()

Erklärung: Die Funktion destruct() muss IMMER an das Ende des Codes außerhalb der While-True-Schleife eingefügt werden. Sie bewirkt, dass das Programm auch wieder korrekt geschlossen werden kann.

drawWords()

Aufruf mit Parametern/Argumenten: drawWords(Position, Text, Schriftgröße, Schriftfarbe)

Beispiel: drawWords( ( 0 , 0 ) , " Hallo ", 69 , ( 255 , 255 , 255 ) )

Erklärung: Die Funktion drawWords() ermöglicht es einem, Text auf dem Bildschirm anzuzeigen. Die angegebenen Koordinaten ist die linke, obere Ecke des Textes. In diesem Beispiel wird "Hallo" oben links in der Bildschirm-Ecke in Schriftgrösse 69 in weiß ausgegeben.

drawCircle()

Aufruf mit Parametern/Argumenten: drawCircle(Mittelpunkt, Radius, Farbe)

Beispiel: drawCircle( ( 50 , 50 ) , 5 , ( 0 , 0 , 255 ) )

Erklärung: Die Funktion drawCircle() ermöglicht es einem, einen Kreis auf den Bildschirm zu zeichen. In diesem Beispiel wird ein blauer Kreis mit dem Mittelpunkt auf den Koordinaten (50, 50) und einem Radius von 5 Pixeln gezeichnet(10 Pixel Durchmesser).

drawEllipse()

Aufruf mit Parametern/Argumenten: drawEllipse(Start-x-Position, Start-y-Position, Breite, Höhe, Farbe)

Beispiel: drawEllipse( 100 , 0 , 200 , 400 , ( 255 , 255 ,0) )

Erklärung: Die Funktion drawEllipse() ermöglicht es einem, eine Ellipse auf den Bildschirm zu zeichen. In diesem Beispiel wird zuerst ein Viereck mit den Koordinaten x1=100, y1=0, x2=x1+200, y2=y1+400 erstellt, woraus anschließend eine Ellipse erstellt wird. Die Ellipse ist gelb.

drawLine()

Aufruf mit Parametern/Argumenten: drawLine(1. Punkt, 2. Punkt, Farbe)

Beispiel: drawLine( ( 0 , 0 ) , ( 337 , 420 ) , ( 255 , 0 , 0 ) )

Erklärung: Die Funktion drawLine() ermöglicht es einem, eine Linie auf den Bildschirm zu zeichen. In diesem Beispiel wird eine rote Linie von ganz oben links bis zu den Koordinaten 337x420 gezeichnet.

drawRect()

Aufruf mit Parametern/Argumenten: drawRect(Start-x-Position, Start-y-Position, Breite, Höhe, Farbe)

Beispiel: drawRect( 250 , 250 , 150 , 100 , ( 0 , 255 , 0 ) )

Erklärung: Die Funktion drawRect() ermöglicht es einem, ein Viereck auf den Bildschirm zu zeichen. In diesem Beispiel wird ein grünes Quadrat mit den Koordinaten x1=250, y1=250, x2=x1+150, y2=y1+100 gezeichnet.

drawTriangle()

Aufruf mit Parametern/Argumenten: drawTriangle(1. Punkt, 2. Punkt, 3. Punkt, Farbe)

Beispiel: drawTriangle( ( 55 , 55 ) , ( 30 , 70 ) , ( 50 , 70 ) , ( 0 , 0 , 255 )

Erklärung: Die Funktion drawTriangle() ermöglicht es einem, ein Dreieck auf den Bildschirm zu zeichnen. In diesem Beispiel wird ein blaues Dreieck mit den angegebenen Koordinaten gezeichnet.

Tutorial

Python und PyGame installieren

1.Schritt: Falls Python 2.7 noch nicht installiert -> das aktuellste Python 2.7 downloaden und installieren

2.Schritt: Falls PyGame noch nicht installiert -> das aktuellste PyGame für Python 2.7 (!!!) downloaden und installieren

Projekt richtig abspeichern

1.Schritt: Ordner anlegen mit dem Namen "Projekt" und QuizFL in diesem Ordner abspeichern

2.Schritt: Jedes neue Projekt muss im Ordner "Projekt" abgespeichert werden - (NICHT in QuizFL)

QuizFL importen

1. Schritt: Neues Python-Dokument öffnen

2. Schritt: Folgendes am Anfang in das Python-Dokument eingeben:

import QuizFL.quizFL

Programm zum Laufen bringen

1. Schritt: Folgendes zum Beispiel in das Python-Dokument hinzufügen:

function = QuizFL.quizFL
function.init(500, 500, False, (1,33,7))

while True:
function.draw()

function.destruct()

Viel Spaß

Nun ist die FL erfolgreich eingerichtet und es können alle Funktionen in vollem Umfang genutzt werden.
